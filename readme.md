# 麦轮小步兵代码
**本工程同时为该机器人云台主控和底盘主控，由app层robot_def.h中宏定义进行切换**
**烧录前务必保证该宏定义与目标主控对应，不能两个同时解注释，烧录错误代码造成的后果未经过评估**

## 机器人基本信息
* 底盘类型：麦克纳姆轮
* 枪管数：1
* 机械负责人：徐至灏

## 仓库说明
本机器人代码为基于2022Framework_F407的步兵机器人，主体由C++开发，C/C++混合编译
